Background Color

Property CSS background berguna untuk menentukan Efek latar belakang pada suatu elemen HTML. Property yang biasa digunakan dalam CSS Background antara lain :

  • latar belakang warna yaitu dengan background-color atau background saja
  • latar belakang gambar yaitu dengan background-image

Sekarang kita akan membuat background color pada elemen body HTML. Pada contoh dibawah ini, ubah warna latar belakang dari #warna menjadi hijau dengan menambahkan properti css background-color:green;
