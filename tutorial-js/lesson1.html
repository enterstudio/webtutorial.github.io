Menampilkan informasi dengan JavaScript

Kali ini kita belajar tentang JavaScript, JavaScript memungkinkan kita untuk memberikan informasi dalam berbagai cara:

  • Memberikan informasi menggunakan kotak peringatan, dengan cara window.alert()
  • Menulis ke output HTML menggunakan document.write()
  • Menulis ke elemen HTML, menggunakan innerHTML
  • Menulis ke browser console, menggunakan: console.log()

Kamu bisa menggunakan kotak peringatan atau alert box seperti ini: Contoh : <script>alert("value");</script> / <script>alert("Halo, Selamat datang di website saya");</script>
